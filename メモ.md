# Docker操作
## パッケージのインストール等
entrypoint.shに記述　←シェルスクリプト

## コンテナ操作

```bash
docker-compose up -d --build ←コンテナ構築

docker-compose down ←コンテナ削除

docker-compose logs -f　←ログ追跡
```
